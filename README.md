## Features

- Background Music Play
- Player in Notification bar
- Play/Pause Song
- Next/Previous Song
- Swipe to change music
- Slider to seek music
- Repeat, Repeat-once, Repeat-off Queue

## Installation

Install the dependencies.

```sh
cd musicplayer
npm install
```
